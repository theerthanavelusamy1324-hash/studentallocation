🎓 Student Allocation Portal

📌 Project Overview

The Student Allocation Portal is a web-based application developed to automate the student allocation process based on eligibility criteria. The system allows administrators to manage student details, allocate students to courses based on their marks and preferences, and view allocation statistics through an interactive dashboard.

✨ Features

- Student Registration
- Course Management
- Course Preference Selection
- Automatic Student Allocation
- Allocated & Not Allocated Student List
- Dashboard with Charts
- Search and Filter
- Responsive User Interface

🛠️ Technologies Used

- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: Java, Spring Boot
- Database: MySQL
- API Testing: Postman
- Build Tool: Maven

📊 Dashboard

- Total Students
- Allocated Students Count
- Not Allocated Students Count
- Course-wise Student Count Chart
- Course-wise Allocation Chart
- Allocation Overview Chart

📂 Modules

- Login
- Student Management
- Course Management
- Allocation Management
- Dashboard
